## 🛠 Repository Structure

* Problems are categorized by **topic** and/or **difficulty level**:

  ```
  leetcode_probleems/
  ├── Arrays/
  ├── Strings/
  ├── LinkedList/
  ├── Trees/
  ├── DynamicProgramming/
  ├── Graphs/
  ```

Each folder contains `.cpp` files with:

* Problem statement (as a comment)
* My solution approach
* Optimized solution (if applicable)

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/<your-username>/leetcode_probleems.git
   ```
2. Navigate into the repo:

   ```bash
   cd leetcode_probleems
   ```
3. Compile & run any file:

   ```bash
   g++ problem.cpp -o output && ./output
   ```
